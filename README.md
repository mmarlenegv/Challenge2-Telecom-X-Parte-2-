## 📌 Propósito del Proyecto
Este repositorio contiene la segunda fase del proyecto **Telecom X**. Tras haber realizado la limpieza inicial de los datos (Parte 1), el objetivo principal de esta fase es **preparar los datos y seleccionar las características (Feature Selection) óptimas para predecir el Churn (cancelación de clientes)** mediante modelos de Machine Learning. 

Se busca identificar qué variables numéricas y categóricas tienen un impacto real y estadísticamente significativo en la decisión de abandono del cliente, eliminando el ruido y la redundancia.
